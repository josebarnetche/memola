# Upsell Playbook — Cuándo y Cómo Proponer el Siguiente Nivel

**Objetivo:** Maximizar valor del cliente a través de servicios adicionales relevantes, sin ser invasivo.

---

## Principio Fundamental

> Upsell solo cuando el cliente tiene un problema real que podemos resolver.
> Nunca upsell por cumplir métricas.

---

## Matriz de Upsell por Servicio Actual

### Desde Website (Level 1)

| Señal de Trigger | Servicio a Proponer | Pitch |
|------------------|---------------------|-------|
| "¿Cómo sé si funciona?" | Analytics GA4 | "Te configuro tracking para que veas visitantes, de dónde vienen, y qué hacen." |
| Bounce rate >60% | Funnel Optimization | "Mucha gente llega pero se va. Podemos optimizar el flujo para que conviertan." |
| "Mi logo es viejo" | Branding | "Aprovechamos el impulso del sitio nuevo para actualizar toda la identidad." |

| "Quiero más tráfico" | Meta Ads | "El sitio está listo. Ahora podemos traer tráfico pago que convierta." |

**Timing:** 2-4 semanas post-entrega del sitio

## Scripts de Upsell

### Apertura Suave
```
"Vi que [observación específica].
Me parece que [servicio] te ayudaría a [beneficio concreto].
¿Te interesa que te cuente más?"
```

### Follow-up Post-Proyecto
```
"Hola, ¿cómo va todo con [proyecto entregado]?

Revisando los datos, noté que [insight específico].
Esto se puede mejorar con [servicio], que costaría [precio].

¿Te interesa conversarlo?"
```

### Respuesta a Señal Directa
```
"Justamente eso lo resolvemos con [servicio].
Es un proyecto de [precio] + IVA, entregamos en [tiempo].

¿Querés que te arme una propuesta?"
```

---

## Timing de Check-ins

| Post-Entrega | Acción |
|--------------|--------|
| 2 semanas | Email: "¿Cómo va todo? ¿Alguna duda?" |
| 4 semanas | Email: Primer check de métricas + posible upsell |
| 8 semanas | Llamada: Review de resultados + propuesta siguiente nivel |
| 12 semanas | Email: Oferta especial si no avanzó |

---

## Red Flags — No Hacer Upsell

- Cliente insatisfecho con proyecto actual
- Pagos pendientes
- Señales de problemas financieros
- Proyecto actual aún no entregado
- Cliente dijo "no" recientemente (esperar 60+ días)

---

## Métricas de Éxito

| Métrica | Target |
|---------|--------|
| Upsell rate (% clientes que compran más) | >30% |
| Time to upsell (días post-proyecto) | 30-60 días |
| Valor promedio de upsell | +50% del proyecto original |
| Satisfacción post-upsell | Igual o mayor que proyecto 1 |

---

## Regla de Oro

> El mejor upsell es cuando el cliente lo pide.
> El segundo mejor es cuando es tan obvio que el cliente dice "¿por qué no me lo dijiste antes?"
