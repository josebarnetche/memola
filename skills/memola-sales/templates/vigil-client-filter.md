## Qualification Scoring — VIGIL Protocol

### The Five Foundation Questions

For each prospect, score 1 point per "SÍ":

1. **¿Puede sobrevivir el tiempo suficiente para que la estructura importe?**
   - Empresa establecida (5+ años): SÍ
   - Startup agro o emprendimiento nuevo: NO

2. **¿El fracaso aquí es erosión, no explosión?**
   - Negocio tradicional, deterioro gradual: SÍ
   - Dependiente de un contrato o temporada: NO

3. **¿La legitimidad es más valiosa que la velocidad?**
   - Reputación importa, confianza de productores: SÍ
   - Necesita resultados inmediatos para sobrevivir: NO

4. **¿La claridad realmente cambiará los resultados?**
   - Estrategia digital impactará ventas/operaciones: SÍ
   - El negocio funciona igual sin digital: NO

5. **¿Esto funciona sin volverse viral?**
   - Crecimiento orgánico, boca a boca, redes locales: SÍ
   - Necesita escala masiva para funcionar: NO

**Scoring:**
- 5/5 = **PROCEED** — Qualified, prioritize
- 4/5 = **PROCEED** — Qualified, proceed
- 3/5 = **CAUTION** — Review concerns, possibly Level 1 only
- 0-2/5 = **DECLINE** — No fit, do not pursue

## Big Agro Qualification Checklist

### Must Have (Eliminatorios)
- [ ] Empresa del sector agropecuario o cadena de valor
- [ ] Facturación estimada > AR$ 50M/año
- [ ] Presencia en NEA (Corrientes, Chaco, Formosa, Misiones, Entre Ríos, Santa Fe Norte)
- [ ] Decision maker identificado o accesible
- [ ] Sin presencia digital profesional actual (o muy desactualizada)

### Bonus Points (Aumentan prioridad)
- [ ] Conexión en la red de José (+2)
- [ ] Competidor directo ya es cliente MEMOLA (+2)
- [ ] Evento o remate próximo (+1)
- [ ] Mencionado en prensa o redes recientemente (+1)
- [ ] Exportan o tienen potencial exportador (+2)
- [ ] Múltiples unidades de negocio (upsell potential) (+1)

### Red Flags (Descalificadores)
- [ ] Historial de no pagar proveedores
- [ ] Committee decision (nadie decide)
- [ ] "Solo quiero cotización para comparar"
- [ ] Expectativa de resultados garantizados
- [ ] Presupuesto < AR$ 300.000
- [ ] Urgencia extrema sin justificación
