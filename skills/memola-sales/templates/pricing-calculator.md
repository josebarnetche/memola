## Fórmulas de Cálculo

### Seña y Final (30/70)
```
Seña = Total × 0.30
Final = Total × 0.70
```

### Milestones (30/40/30)
```
M1 = Total × 0.30
M2 = Total × 0.40
M3 = Total × 0.30
```

### IVA
```
IVA = Subtotal × 0.21
Total con IVA = Subtotal × 1.21
```

## Ejemplo de Cotización

**Cliente quiere:**
- Website multi-page: AR$ 1.000.000
- Logo + manual: AR$ 500.000
- Meta Ads setup: AR$ 1.000.000 (sin IVA, ya que el precio es IVA incluido)

**Cálculo:**
```
Subtotal web + logo = AR$ 1.500.000
IVA = AR$ 315.000
Meta Ads = AR$ 1.210.000 (IVA incl.)
TOTAL = AR$ 3.025.000

Seña (30%): ~AR$ 907.500
Milestone (40%): ~AR$ 1.210.000
Final (30%): ~AR$ 907.500
```
